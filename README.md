## How to use

- Clone the repository

```bash
git clone git@github.com:lucasdximenes/recipe-app.git
```

- Enter the folder

```bash
cd recipe-app
```

- Install the dependencies

```bash
npm install
```

- Run the project

```bash
npm start
```

- Open the browser and access the address

```bash
http://localhost:3000/
```
